Message-ID
This line deletes the single USENET message identified by the Message-ID indicated in the first and only input field. The Message-ID must be entered complete with all its parts, including the at ('@'), the start ('<') and end ('>') characters; wildmats are not allowed.
Example: with first field set to <r50rmn$p52$1@gioia.aioe.org> this line deletes the message identified by Message-ID <r50rmn$p52$1@gioia.aioe.org>
Range
This option deletes all messages sent to the newsgroup indicated by the first input field and still kept on the server that have a progressive serial number between the minimum value indicated by the second input field and the maximum indicated by the third. Numbers must be entered in decimal integer format and must be positive.
Warning: all messages with a serial number between the minimum (second input field) and the maximum (third field) are deleted without exception.
Example: with first field set to local.test, second set to 1 and third to 999, this command cancels all messages sent to the newsgroup 'local.test' that have got a serial number between 1 and 999.
Thread
This command deletes the USENET message identified by the Message-ID indicated in the first and only input field as well as all the messages that reply to it. In other words, it deletes the message identified by the message-id entered in the input field and all the messages that contain it within the argument of header References.
Warning: in the case of very long threads, some newsreader insert only the last three or four Message-IDs into the value of the References header, therefore not all the messages belonging to a thread could be deleted using this command because in some of them the Message-ID of the post that originated the thread may not be indicated.
Example: if <r50rmn$p52$1@gioia.aioe.org> is inserted in the first and only input field, the USENET message identified by <r50rmn$p52$1@gioia.aioe.org> as Message-ID is deleted as well as all posts that includes <r50rmn$p52$1@gioia.aioe.org> inside the argument of References header
Sender
This command deletes all messages posted by the same sender that sent the message identified by the Message-ID indicated in the input field.
Warning: this command deletes only the messages sent on the same group of the message whose message-id is inserted in the input field and only if the content of the From header is perfectly equal.
Example: if <r5nej6$irh$1@dont-email.me> is inserted into the only input field, all messages sent to it.discussioni.auto by 'Hugo Stiglitz <HugoStiglitz@giovannona.coscialunga.com>' will be cancelled because that message was sent to this group by this author.
Header
Inside the group set by the first input box, all messages whose header identified by the second field matches the value shown by the third field are deleted.
Example: local.test (first field), User-Agent (second field), Pan (third field) will cancel all messages inside local.test whose User-Agent header matches "Pan".
Warning: third field, header value, is case sensitive so Pan doesn't match pan
Body
Delete all messages sent to the group shown by the first field that match the regular expression inserted in the second field. Pattern is case insensitive.
Warning: this operation is expansive in terms of CPU time if the messages in the group are more than a few thousands
